Presentatie

Hier leg ik kort uit wat ik heb geleerd en gedaan bij elk van de projecten.

1. UTP Kabel Aansluiten

Bij dit project moest ik zelf een UTP-kabel in elkaar steken. Ik leerde hoe ik de kabel correct moest strippen, de draadjes op de juiste manier moest ordenen en de RJ45-connectoren moest krimpen. Het was een praktisch proces waarbij ik alles zelf moest doen, van het strippen tot het aansluiten van de kabels, wat essentieel is voor het opzetten van betrouwbare netwerkverbindingen.

2. HYPER-V Server Configureren

In dit project leerde ik hoe ik een Hyper-V-server moest configureren, inclusief het toevoegen van virtuele LAN- en WLAN-netwerken en netwerkkaarten. Ik moest een virtuele infrastructuur opzetten om meerdere virtuele machines (VM's) te creëren op één fysieke server. Dit hielp me bij het begrijpen van het optimaliseren van serverresources en het verhogen van de flexibiliteit van een IT-omgeving.

3. Linux Server Configureren

Voor de Linux server leerde ik hoe ik SSH-keys moest configureren zodat ik zonder telkens een wachtwoord in te voeren kon inloggen. Ik heb de basisprincipes van het configureren van een Linux-server geleerd, zoals het instellen van gebruikers, rechten en services. Dit project hielp me bij het opzetten van een robuuste en veilige serveromgeving.

4. Windows Server Beheer

Bij dit project leerde ik hoe ik een Windows-server moest beheren. Ik leerde alles over gebruikersrechten, hoe ik gebruikersaccounts moest aanmaken en hoe ik de rechten van gebruikers kon bepalen. Dit gaf me een goed begrip van het beheren van Windows-serveromgevingen en het configureren van netwerkvoorzieningen.

5. Wireshark

Met Wireshark heb ik gewerkt aan een grote opdracht waarin ik netwerkpakketten moest monitoren die binnenkomen en naar buiten gaan. Ik heb leren analyseren wat er precies door het netwerk beweegt en dit heeft me geholpen bij het identificeren van netwerkproblemen en het verbeteren van de netwerkprestaties door inzicht te krijgen in de pakketstromen.

6. Packet Tracer

Bij Packet Tracer heb ik aan een grote opdracht gewerkt waarbij we alles zelf moesten plaatsen in een groot gebouw. Dit project omvatte het plaatsen van racks, routers en alles wat nodig was om een netwerk in een echte werkomgeving op te zetten. Ik leerde alles over het ontwerpen van netwerkomgevingen en hoe verschillende netwerkonderdelen correct moesten worden geconfigureerd en getest om een solide netwerktopologie te creëren.
