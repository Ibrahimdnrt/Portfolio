Presentatie

Hier leg ik kort uit informatie over onze projecten

1. UTP kabel Aansluiten

Beschrijving: Leer stap voor stap hoe je een UTP-kabel correct aansluit, inclusief het krimpen van de RJ45-connectoren. Dit project is essentieel voor het opzetten van betrouwbare netwerkverbindingen.

2. HYPER-V server configureren

Beschrijving: Zet een virtuele infrastructuur op om meerdere virtuele machines (VM's) te creëren op één fysieke server. Dit helpt bij het optimaliseren van serverresources en verhoogt de flexibiliteit van je IT-omgeving.

3. Linux server configureren

Beschrijving: Leer de basisprincipes van het configureren van een Linux-server, inclusief het instellen van gebruikers, rechten en services. Dit project helpt bij het opzetten van een robuuste en veilige serveromgeving.

4. Windows server beheer

Beschrijving: Beheers de dagelijkse beheeractiviteiten van een Windows-server, zoals het opzetten van Active Directory, het beheren van gebruikersaccounts en het configureren van netwerkvoorzieningen.

5. Wireshark

Beschrijving: Gebruik Wireshark om netwerkverkeer te monitoren en te analyseren. Dit project helpt bij het identificeren van netwerkproblemen en het optimaliseren van netwerkprestaties door inzicht in pakketstromen te krijgen.

6. Packet tracer

Beschrijving: Maak gebruik van Cisco Packet Tracer om een virtuele netwerkomgeving te ontwerpen en te simuleren. Leer hoe je verschillende netwerkonderdelen configureert en test om solide netwerktopologieën op te bouwen.
